Minesweeper project

Key Features
1. 숫자 색 표시 기능
- config.py를 통해 각 숫자마다 다른 색을 적용 할 수 있음.
- 지뢰가 없는 칸(0)은 숫자가 표시되지 않음.

- <img width="447" height="774" alt="image" src="https://github.com/user-attachments/assets/2f4015c1-c71c-4c26-aa83-232ef1f0d4c2" />


2. 난이도 설정 기능
- Easy / Normal / Hard 난이도를 제공함.
  (1번 키는 Easy, 2번 키는 Normal, 3번 키는 Hard)
- 난이도에 따라 보드 크기와 지뢰 수가 변경됨.
- R 키를 눌러 게임 재시작 할 시 난이도가 유지됨.
- 난이도 변경에 따라 화면 크기가 변화됨.
- 난이도 선택 시 화면에 1: Easy 2: Normal 3: Hard 가 표기됨.

3. 힌트 기능
- 지뢰가 없는 칸만 선택하여 열림.
- 이미 열린 칸은 제외하고 열림.
- 힌트 사용 횟수는 3회로 제한됨.
- 화면에서 힌트가 몇 개 남았는지 확인할 수 있음.

4.하이 스코어 저장 및 표출 기능
- 최단 시간을 기록 후 저장함.
- 게임 종료 후 기록 출력함.

5. 타이머 표기 기능
- 게임 시작 시 타이머가 시작됨.
- 게임 종료 시 타이머가 정지됨.
- 화면에 실시간으로 타이머가 표시됨.
